# Maple-Check
Maple-Check is a simple application built with React that helps users of the popular game "Maplestory" track daily and weekly tasks, as well as providing helpful information about the game.

## Features
- Track daily and weekly tasks with a convenient to-do list.
- View helpful information about the game, including daily/weekly reset timers, guides and calculators.
- Customize your to-do list with the tasks that are most important to you by adding characters to track and customize their specific tasks.

## Requirements
Node.js and npm (included with Node)

## Installation
1. Clone this repository onto your local machine using `git clone https://github.com/Giuzus/maple-check-app.git`
2. Navigate to the cloned directory using `cd maple-check`
3. Install the required dependencies using `npm install`
4. Start the development server using `npm start`
5. The application should now be running at http://localhost:3000

## Usage
To add a character and their associated tasks to your to-do list, simply click the "Add Character" button and enter the character name. The application will then display a list of pre-created tasks specific to that character. To mark a task as complete, click the checkbox next to it.
